# Coyote-rs

Create `HTML | XML | DOCUMENTS | SCENES` in Rust with function components.

## Components

Create documents with `coyote` [components](./coyote/README.md).

## Html

Render components as [html](./coyote_html/README.md) with `coyote_html`.
